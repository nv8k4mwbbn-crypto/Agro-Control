AGRO CONTROL PWA

Esta carpeta contiene una aplicación web instalable.

ARCHIVOS PRINCIPALES
- index.html
- styles.css
- app.js
- manifest.webmanifest
- service-worker.js

CÓMO PUBLICAR GRATIS EN GITHUB PAGES
1. Crea o abre tu repositorio en GitHub.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio.
3. En GitHub entra en Settings > Pages.
4. En "Build and deployment" selecciona "Deploy from a branch".
5. Elige la rama main y la carpeta /root.
6. Guarda y espera a que aparezca la dirección pública.

INSTALAR EN IPHONE
1. Abre la dirección pública con Safari.
2. Pulsa Compartir.
3. Pulsa "Añadir a pantalla de inicio".
4. Confirma con "Añadir".

Los datos se guardan en el navegador del dispositivo. Usa periódicamente
Ajustes > Descargar copia de seguridad.
